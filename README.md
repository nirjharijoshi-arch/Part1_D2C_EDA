# D2C Customer Churn Intelligence – Part 1

## Objective

This project performs an exploratory data analysis and data quality assessment for a Direct-to-Consumer (D2C) business experiencing customer churn.

## Datasets Used

- customers.csv
- orders.csv
- support_tickets.csv
- web_events_snapshot.csv
- intervention_history.csv
- churn_labels.csv
- rfm_modeling_snapshot.csv

## Key Findings

- Total customers analyzed: 2,400
- Overall churn rate: 46.96%
- Customers with high recency appear substantially more likely to churn.
- Lower engagement levels are associated with increased churn risk.
- Support-ticket activity may be correlated with churn.
